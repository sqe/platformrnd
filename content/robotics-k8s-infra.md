Production-grade platform for cloud/edge robotics integration. Features KubeEdge for edge computing, ROS2 middleware, ArgoCD for GitOps deployment, multi-domain DDS for distributed systems, eBPF-based networking with Cilium for UDP multicast discovery, CLI tools for edge node simulation and lifecycle management, and real-time metrics collection.
